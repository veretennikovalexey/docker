Hello, World!

Welcome to my simple website powered by Docker!

Конечно! Давайте разберём, как с помощью Docker можно создать простой веб-сайт.

Представьте, что Docker — это как коробка, в которую вы можете положить всё, что нужно вашему сайту, чтобы он работал, и эта коробка будет одинаково работать на любом компьютере.

Вот шаги, чтобы создать простой веб-сайт с помощью Docker:

  1. Установите Docker:
    • Сначала нужно установить Docker на ваш компьютер. Это можно сделать, скачав Docker Desktop с официального сайта Docker.
  2. Создайте папку для вашего проекта:
    • Создайте новую папку на вашем компьютере, например, my-website.
  3. Создайте HTML файл:
    • Внутри папки my-website создайте файл index.html. Это будет ваш веб-сайт. Например, в index.html можно написать: 
      
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Simple Website</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Welcome to my simple website powered by Docker!</p>
</body>
</html>
  4. Создайте Dockerfile:
    • В той же папке my-website создайте файл с именем Dockerfile (без расширения). В этом файле напишите: 
      
FROM httpd:2.4
COPY ./index.html /usr/local/apache2/htdocs/
    • Это говорит Docker использовать образ Apache (это программа, которая показывает ваш сайт) и скопировать ваш index.html в нужное место.
  5. Постройте Docker образ:
    • Откройте терминал (или командную строку) и перейдите в папку my-website.
    • Введите команду: 
      
docker build -t my-simple-website .
    • Эта команда создаст Docker образ с вашим сайтом.
  6. Запустите контейнер:
    • Теперь, чтобы увидеть ваш сайт в действии, введите команду: 
      
docker run -d -p 8080:80 my-simple-website
    • Эта команда запустит ваш сайт в контейнере и сделает его доступным на вашем компьютере по адресу http://localhost:8080.

Теперь, открыв браузер и перейдя по адресу http://localhost:8080, вы увидите ваш простой веб-сайт!

Docker позволяет легко упаковать и запустить ваш сайт, не беспокоясь о том, как он будет работать на других компьютерах. Это как магическая коробка, которая всегда работает одинаково!
